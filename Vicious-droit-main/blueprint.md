# **App Name**: DroitForge

## Core Features:

- Source Asset Management: Sophisticated drag-and-drop interface for managing Android project files, source code, and binary resources.
- Project Metadata Forge: Precise controls for defining application package identifiers, semantic versioning, and SDK compatibility targets.
- AI Permission Analyst: An AI tool that scans uploaded source code to automatically suggest necessary Android manifest permissions based on the API calls detected.
- Interactive Dependency Registry: A categorized library of popular mobile frameworks allowing developers to architect their app with a single click.
- Virtual Build Pipeline: Real-time, sequenced visual console providing feedback during the compilation and manifest generation process.
- Cloud Package Storage: Secure storage for your build configurations and manifest exports using a cloud database solution.
- Package Manifest Generator: Instant generation and export of APK manifest XML and build configuration bundles.

## Style Guidelines:

- Primary color: Electric Cyan (#00FFD5) to evoke a sense of high-tech precision. Background: Deep Onyx (#121615) providing a high-contrast foundation. Accent: Bright Azure (#009DFF) for secondary interactive elements.
- Headline font: 'Space Grotesk', a tech-centric sans-serif for a computerized feel. Body font: 'Inter', providing clean, objective readability for technical logs and labels.
- Minimalist monolinear icons with subtle glowing outer borders when active, mimicking high-tech HUD systems.
- A modular dashboard with clear structural boundaries and generous spacing, following a strict technical grid layout.
- Sequenced scanning effects for AI file analysis and 'terminal typing' effects for build logs.
